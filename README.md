# advent-of-code

[Advent of Code](https://adventofcode.com/) solutions.

## Run

```sh
export AOC_SESSION=<value-of-session-cookie-in-browser>
```

### 2015 (C)

```sh
make -C 2015
./2015/solutions.x <problem> <part>
```

### 2016 (C++)

```sh
make -C 2016
./2016/solutions.x <problem> <part>
```

### 2022 (Python)

```sh
python 2022/solutions.py <problem> <part>
```
